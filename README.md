# ProxmoxVETools
个人使用 Proxmox VE 工具脚本

# 添加 Btrfs 存储脚本使用说明

下载脚本

```shell
curl -OL https://raw.githubusercontent.com/lwlf/ProxmoxVETools/main/add-btrfs-storage.sh && chmod +x add-btrfs-storage.sh
```

使用脚本

```shell
./add-btrfs-storage.sh
```
